# Installation de la librairie MPU6050
1. Dans Arduino IDE : Sketch -> Include library -> Add .ZIP library
2. Sélectionner l'archive MPU6050.zip

# Branchements
Le capteur se connecte avec le protocole I2C à l'arduino

| Capteur | Arduino |
|---------|---------|
| VCC     | 3.3V    |
| GND     | GND     |
| SCL     | SCL     |
| SDA     | SDA     |
